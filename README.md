# Library Manager

The Library Manager is a web application built using Flask and SQLAlchemy. It allows users to manage books and authors within a library system.

## Features

- **View Books**: Browse through a list of books, filter by title and author.
- **Manage Books**: Add new books, update existing books, and delete books from the library.
- **Manage Authors**: Add new authors, delete authors (and their associated books).
- **Export Books**: Generate and download a CSV file containing all books and their authors.

## Usage

### Home
- View all books
- Filter by title and author
- Delete books
- Export a list of books

### Manage Book
- Add new books
- Update existing books

### Manage Author
- Add new authors
- Delete authors
